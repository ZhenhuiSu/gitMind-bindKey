### gitMind-ext（GM扩展插件）
##### 功能：
- **左移节点（Alt+Left）**
- **右移节点（Alt+Right）**
- **渲染markdown（Alt+M）**<br>
在新标签页渲染备注以[markdown]/n(回车)开头的内容
- **自定义原生快捷键**<br>
点击快捷键按钮，在具体功能输入自定义快捷键，保存即可。恢复原生快捷键时清空输入框并保存即可。<br>
自定义的快捷键会覆盖不会改变原生快捷键，但与原生/浏览器快捷键出现快捷键冲突时，以自定义快捷键为准
